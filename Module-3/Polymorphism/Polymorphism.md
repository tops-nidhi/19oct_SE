**Polymorphism**
- One name many form

Types:
1. Static (compile time) :-
    1. Overloading
        1. Function overloading
          - same name , same class  but different parameter
          
        2. Operator Overloading
  
    2. Overriding
        1. Function overriding
          - same name, same parameter but different class
       
2. Dynamic (Run time) :-
   - Virtual polymorphism
